## 긴 설명

```
"아, 앉아서 가고 싶다. 제발."

대중교통으로 인생의 1/3을 보내는 경기도민이라면,
한 번쯤 마음속으로 외쳐봤을 거예요.

대중교통에서 흘러가는 그 시간만큼은 우리에게도 소중하잖아요.
앉아서 가면 자거나, 책을 읽거나, 잠깐 눈이라도 붙일 수 있어요.
저는 경기도민에게 그 작은 쉼을 드리고 싶었습니다.

출퇴근은 매일의 작은 전쟁.
빈자리 하나 보이면 그게 그날의 작은 승리잖아요.

시티드는 서울교통공사 실시간 데이터와 SK PUZZLE 칸별 혼잡도로
"내가 탈 열차에서 어느 칸이 가장 한산할지" 미리 알려드립니다.

이 빈 자리가, 당신에게 작은 위로가 되기를.

▣ 핵심 기능
• 1~10호차 실시간 칸별 혼잡도
• 4호선·2호선·9호선 환승 최적 칸 안내
• 자주 가는 경로 한 번에 — 빠른 출발
• 위치 기반 가까운 역 자동 선택

▣ 데이터 출처
• 서울교통공사 실시간 도착정보
• SK PUZZLE-SUBWAY 칸별 혼잡도 통계
• 코레일 KRIC 역사 정보

▣ 만든 이유
앞에 앉은 사람을 빤히 쳐다보며
"저 사람 언제 내리지...?" 간절했던 적, 다들 있잖아요.
그 마음에서 시작했어요.

대중교통 안의 시간이 조금 더 편안하시길 바랍니다.

만든 사람: 프로젝트윤
문의: [이메일]
```
