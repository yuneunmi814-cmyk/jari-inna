개인정보처리방침

최종 수정일: 2026년 5월 13일 · 시행일: 2026년 5월 13일

요약

· 시티드는 회원가입이 없습니다.

· GPS 위치는 “현재 위치에서 가까운 역”을 찾을 때만 일시적으로 사용하고 저장하지 않습니다.

· 즐겨찾기·최근 선택 같은 설정값은 이용자의 기기 안에만 저장됩니다.

· 어떠한 정보도 외부 서버로 전송되거나 제3자에게 제공되지 않습니다.

1. 개요

시티드(이하 “서비스”)는 수도권 광역철도 이용자가 자리에 앉아 갈 가능성이 높은 칸을 추천받기 위한 모바일 앱입니다. 본 방침은 서비스 이용 과정에서 처리되는 개인정보의 항목·목적·보관 방법·이용자 권리를 안내합니다.

2. 수집하는 정보

서비스는 회원가입을 요구하지 않으며, 이름·이메일·전화번호 등 개인을 식별할 수 있는 정보를 수집하지 않습니다. 다음 항목만 기능 수행에 필요한 범위에서 처리됩니다.

항목목적처리 방식
위치 정보 (GPS 좌표) 현재 위치에서 가장 가까운 역을 찾아 출발역으로 자동 제안 이용자가 권한을 허용한 시점에만 일시적으로 조회. 기기·서버 어디에도 저장하지 않음
이용자 설정값 다음 실행 시 동일한 환경 유지 이용자 기기 내 AsyncStorage(로컬 저장소)에만 저장. 외부 전송 없음

기기 내부에 저장되는 설정값은 구체적으로 다음과 같습니다.

  • @jari-inna/selected-station — 마지막으로 선택한 출발역 이름
  • @jari-inna/departure-line — 마지막으로 선택한 출발 호선
  • @jari-inna/favorites — 이용자가 즐겨찾기에 추가한 역 목록
  • @jari-inna/favorite-routes — 이용자가 즐겨찾기에 추가한 경로 목록

3. 위치 정보 처리에 관한 안내

  • 위치 권한은 “앱 사용 중에만 허용”·“이번만 허용” 등 OS가 제공하는 옵션으로 이용자가 직접 선택합니다.
  • 권한을 허용하지 않아도 서비스의 다른 기능(역 직접 선택, 혼잡도 보기, 환승 안내 등)은 정상 이용할 수 있습니다.
  • 조회한 좌표는 가까운 역을 계산하는 즉시 메모리에서 폐기되며, 디스크나 외부 서버에 기록되지 않습니다.
  • 이용자는 언제든 OS 설정 → 앱 권한에서 위치 권한을 회수할 수 있습니다.

4. 보유 및 이용 기간

  • 위치 정보: 보관하지 않음 (조회 즉시 폐기)
  • 설정값: 이용자가 앱을 삭제하거나 앱 데이터를 초기화할 때까지 기기 내부에 보관

5. 제3자 제공 및 처리위탁

서비스는 이용자의 어떠한 정보도 제3자에게 제공하거나 외부에 위탁하지 않습니다. 또한 광고 식별자(ADID/IDFA) 수집, 사용 분석 SDK, 크래시 리포트 등 외부 전송이 발생하는 서드파티 도구를 사용하지 않습니다.

6. 외부 데이터 소스

열차 운행 정보·역 정보는 한국철도공사(Korail) 및 공공데이터포털(data.go.kr) 등에서 제공하는 공개 API를 이용해 이용자의 기기에서 직접 조회합니다. 이 과정에서 이용자를 식별할 수 있는 정보는 함께 전송되지 않습니다.

7. 이용자의 권리

  • 위치 권한 허용/회수: OS 설정에서 언제든 변경 가능 (앱 내 설정 → 위치 → 시스템 설정 열기 로 바로 이동)
  • 저장된 설정값 삭제: 앱 내 설정 → 데이터 관리 에서 즐겨찾기·최근 선택·전체 데이터를 항목별로 언제든 삭제 가능. OS의 앱 데이터 초기화나 앱 삭제로도 즉시 제거됩니다.

8. 안전성 확보 조치

서비스는 개인정보를 수집·전송하지 않으므로 외부 유출 위험을 구조적으로 차단합니다. 기기 내부에 저장되는 설정값은 OS가 제공하는 앱 샌드박스 안에서만 접근 가능합니다.

9. 만 14세 미만 아동

서비스는 만 14세 미만 아동을 대상으로 별도의 개인정보를 수집하지 않습니다.

10. 처리방침 변경

본 방침은 법령·서비스 변경에 따라 개정될 수 있습니다. 개정 시 본 페이지의 최종 수정일을 갱신하며, 중요한 변경이 있는 경우 앱 내 공지 등을 통해 안내합니다.
