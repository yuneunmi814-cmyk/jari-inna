개인정보처리방침

시행일: 2026년 5월 22일
한눈에 보기
  • 회원가입이 없어요. 이름·이메일·전화번호 같은 건 아예 받지 않아요.
  • GPS는 “가까운 역 찾기”에만 잠깐 쓰고 어디에도 저장하지 않아요.
  • 즐겨찾기·최근 선택값만 이용자 기기 안에 저장돼요.
  • 외부로 보내는 정보도, 제3자에게 주는 정보도 없어요.
  • 설정 화면에서 언제든 데이터를 직접 삭제할 수 있어요.

1이 방침이 다루는 것

시티드(이하 “서비스”)는 수도권 광역철도 이용자가 “자리에 앉아 가기 좋은 칸”을 추천받기 위한 모바일 앱이에요. 본 방침은 서비스를 쓸 때 어떤 정보가 어디에서 어떻게 처리되는지를 정리해 둔 문서예요.

2받는 정보 (수집 항목)

서비스는 가입 절차가 없고, 이름·이메일·전화번호 같은 개인을 식별할 수 있는 정보를 받지 않아요. 딱 두 가지만 기능 수행에 필요한 만큼 처리돼요.

항목목적처리 방식
위치 정보 (GPS 좌표) 현재 위치에서 가장 가까운 역을 찾아 출발역으로 자동 제안 이용자가 권한을 허용한 시점에만 메모리에서 잠깐 사용. 계산이 끝나면 즉시 폐기. 디스크·서버 어디에도 저장 안 함.
이용자 설정값 다음 실행 시 동일한 환경 유지 이용자 기기 내 로컬 저장소(AsyncStorage)에만 저장. 외부 전송 없음.

기기 내부에 저장되는 설정값은 정확히 아래 네 가지뿐이에요.

  • @jari-inna/selected-station — 마지막으로 선택한 출발역 이름
  • @jari-inna/departure-line — 마지막으로 선택한 출발 호선
  • @jari-inna/favorites — 즐겨찾기에 추가한 역 목록
  • @jari-inna/favorite-routes — 즐겨찾기에 추가한 경로 목록 (출발↔도착)

3위치 정보에 대한 안내

  • 위치 권한은 “앱 사용 중에만 허용”, “이번만 허용” 같은 OS 기본 옵션 중 이용자가 직접 선택해요.
  • 권한을 허용하지 않아도 다른 기능(역 직접 선택, 혼잡도 보기, 환승 안내 등)은 그대로 쓸 수 있어요.
  • 조회한 좌표는 가장 가까운 역을 계산하는 그 순간에만 메모리에 머물고, 계산이 끝나면 바로 사라져요.
  • OS 설정 → 앱 권한에서 언제든 위치 권한을 회수할 수 있어요. 앱 내 설정 → 위치 → 시스템 설정 열기 로 바로 이동할 수도 있어요.

4보관 기간

  • 위치 정보: 보관하지 않음 (조회 즉시 폐기)
  • 설정값: 이용자가 앱을 지우거나 직접 삭제할 때까지 기기 안에 보관

5제3자 제공 · 외부 전송

서비스는 이용자의 어떠한 정보도 외부로 보내지 않아요. 제3자 제공·위탁 모두 없어요. 광고 식별자(ADID/IDFA) 수집, 사용 분석 SDK, 크래시 리포트 등 백그라운드에서 정보를 전송하는 서드파티 도구도 사용하지 않아요.

6외부 공공 데이터

열차 운행·역 정보는 한국철도공사(Korail), 공공데이터포털(data.go.kr), 서울교통공사·국가철도공단 등 공공 기관이 제공하는 공개 API를 이용자의 기기에서 직접 조회해서 보여줘요. 이 과정에서 이용자를 식별할 수 있는 정보는 함께 전송되지 않아요.

7이용자의 권리 (데이터 삭제 포함)

  • 앱 내에서 직접 삭제: 설정 → 데이터 관리 에서 언제든 항목별로 지울 수 있어요.
    • 즐겨찾기 초기화 — 즐겨찾기 역과 경로를 모두 삭제
    • 최근 검색 기록 삭제 — 마지막 선택한 출발역·호선을 삭제
    • 모든 데이터 초기화 — 위 네 키 전부 한 번에 삭제
  • OS에서 일괄 삭제: 시스템 설정의 “앱 데이터 초기화” 또는 앱 자체 삭제 시 모든 로컬 데이터가 자동으로 함께 제거돼요.
  • 위치 권한 변경: OS 설정에서 언제든 허용/거부를 바꿀 수 있어요.

8안전성 확보 조치

서비스는 개인정보를 외부로 보내지 않는 구조라서, 유출 위험을 구조적으로 차단해요. 기기 안에 저장되는 설정값도 OS가 제공하는 앱 샌드박스 안에서만 접근할 수 있어요.

9만 14세 미만 아동

서비스는 만 14세 미만 아동을 대상으로 별도의 개인정보를 수집하지 않아요. 회원가입 자체가 없어요.

10방침 변경 안내

본 방침은 법령·서비스 변경에 따라 개정될 수 있어요. 개정이 있을 때는 이 페이지의 시행일을 갱신하고, 중요한 변경이 있는 경우 앱 내 공지 등으로 알려드려요.
